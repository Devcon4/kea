Web Accessibility Fundamentals

March 22, 2026

Web accessibility ensures that websites are usable by everyone, including people with disabilities. The Web Content Accessibility Guidelines (WCAG) provide a shared standard for web accessibility.

Key Principles

  • Perceivable — Information must be presentable in ways users can perceive (alt text, captions, contrast).
  • Operable — Interface components must be operable (keyboard navigation, enough time, no seizures).
  • Understandable — Information and UI operation must be understandable (readable, predictable, input assistance).
  • Robust — Content must be robust enough for assistive technologies to interpret.

Common Issues

The most common accessibility issues include missing alt text on images, poor color contrast, missing form labels, and lack of keyboard navigation support. Automated tools like Kea can detect many of these issues.
