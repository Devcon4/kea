# Operator Instructions

## Framework

- Go, Kubebuilder v4, controller-runtime.
- CRDs: `AgentPool`, `TargetResource`, `TestPlan` under `chaos.kea.dev/v1alpha1`.

## Patterns

- Level-based reconciliation. Reconcile loops are idempotent.
- Use `controllerutil.SetControllerReference` for owner refs.
- Use finalizers for cleanup.
- Status subresource for tracking state.

## Testing

- envtest + Ginkgo/Gomega. No real cluster needed for unit tests.
- Test reconcile logic: create CR → verify pods created, scale up/down, cleanup on delete.

## Style

- Standard Go conventions. `gofmt` / `goimports`.
- Error handling: explicit `if err != nil` checks, wrap errors with `fmt.Errorf("context: %w", err)`.
- No magic. Explicit is better than implicit.
